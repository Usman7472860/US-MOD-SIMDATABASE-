## 🚀 Description

**SIM Database** ek premium web tool hai jo Pakistan ke SIM numbers ki information lookup karne ke liye banaya gaya hai. Yeh tool `wasif-ali` ki API use karta hai aur real-time data fetch karta hai.

## ✨ Features

- 🔍 **Live SIM Lookup** — Real-time number search
- 👤 **Owner Card** — Verified admin display with photo
- 📊 **Full Record Display** — Name, CNIC, Mobile, Network, Address
- 🌐 **No Zoom** — Mobile-optimized, zoom disabled
- ⚡ **Fast Search** — Enter press ya button se search
- 🎨 **Premium Dark UI** — Neon green + dark theme

## 🔧 How to Use

1. `simdatabase.html` file download karo
2. Browser mein open karo (Chrome / Firefox)
3. Number enter karo (e.g. `03001234567`)
4. **SEARCH** button dabao ya Enter press karo
5. Result screen par show hoga ✅

## 🌐 API Info

```
Endpoint: https://sim-info-api.wasif-ali.workers.dev/?search=<number>

Response Format:
{

  "count": 1,
  "records": [
    {

      "mobile": "...",
      "cnic": "...",

      "network": "..."
    }
  ]
}
```
